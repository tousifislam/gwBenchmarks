## Finite-Size Balance-Law SPA Notes

- Implemented `h_of_f(...)` in `candidate_waveform.py` using only the source-packet formulas.
- Used detector-frame quantities:
  - `M_sec = Mc * MSUN_SEC / eta^(3/5)`
  - `Mc_sec = Mc * MSUN_SEC`
  - `dL_sec = dL * MPC_SEC`
  - `v = (pi M_sec f)^(1/3)`.
- Built the balance law with:
  - binding energy `e(v) = E(v)/M = -eta v^2/2 [1 + E_SS_SIM(v) v^4 + E_SSS_SIM v^7]`
  - radiated flux at infinity `F_infty`
  - absorbed flux `dotM` including the benchmark `18 H0 eta^2 v^18` convention.
- Used:
  - `F_tot = F_infty + dotM`
  - `dt/dv = -M_sec (de/dv) / F_tot`
  - `df/dt = [3 v^2 / (pi M_sec)] / (dt/dv)`.
- SPA amplitude:
  - `A_N = sqrt(5/24) Mc_sec^(5/6) pi^(-2/3) f^(-7/6) / dL_sec`
  - `A = A_N * sqrt(dfdt_N / dfdt)`
  - `dfdt_N = 96 eta v^11 / (5 pi M_sec^2)`.
- Phase convention:
  - integrated `dt/df` and `dphi/df = 2 pi f dt/df` from highest valid frequency downward
  - anchored with `t(f_max)=tc`, `phi(f_max)=phic`
  - used `psi(f) = 2 pi f t(f) - phi(f) - pi/4`
  - returned `h(f) = A(f) exp[-i psi(f)]`.
- Enforced prompt cutoffs exactly:
  - `h(f)=0` for `f < f_low` and `f >= f_cut`
  - logistic taper `W(f)=1/(1+exp((f-f_cut)/sigma))` inside valid band.
