# arXiv:2410.00294 Relevant Formulas

This compact formula sheet collects the Appendix-A ingredients needed for an
inspiral-only finite-size waveform benchmark. It is not project source code.

The physics idea is the following. The paper describes finite-size effects
through the binding energy, the flux radiated to infinity, and the absorbed
flux:

```text
-F_infty(v) - dotM(v) = dE/dt.
```

For the benchmark, use the formulas below to construct a balance-law
frequency-domain waveform. Keep the point-particle baseline Newtonian, include
the spin-induced finite-size pieces printed below, use the effective leading
Love-number term specified below, and use the doubled BBH convention for the
spin-independent absorbed-flux coefficient.

Use `G = c = 1`, detector-frame masses, and

```text
MSUN_SEC = 4.925491025543576e-6
MPC_SEC  = 3.0856775814913673e22 / 299792458
M_sec    = Mc * MSUN_SEC / eta^(3/5)
dL_sec   = dL * MPC_SEC
v        = (pi M_sec f)^(1/3)
delta    = sqrt(1 - 4 eta)
chi_s    = (chi1 + chi2) / 2
chi_a    = (chi1 - chi2) / 2
```

## Binding Energy

Appendix A writes the binding energy as

```text
E(v) = -M eta v^2/2 [
    E_NS(v) + E_SO(v) v^3 + E_SS(v) v^4
    + E_SSS(v) v^7 + E_Love(v) v^10
].
```

For this compact benchmark use

```text
e(v) = E(v)/M
     = -eta v^2/2 [1 + E_SS_SIM(v) v^4 + E_SSS_SIM v^7].
```

The spin-induced quadrupole contribution is

```text
E_SS_SIM(v) =
chi_a chi_s (-delta^2 kappa_a/2 - delta kappa_s - kappa_a/2)
+ chi_s^2 (-delta^2 kappa_s/4 - delta kappa_a/2 - kappa_s/4)
+ chi_a^2 (-delta kappa_a/2 + eta kappa_s - kappa_s/2)
+ v^2 [
    chi_a^2 (25 delta eta kappa_a/12 - 35 delta kappa_a/12
             - 5 eta^2 kappa_s/6 + 95 eta kappa_s/12
             - 35 kappa_s/12)
  + chi_a chi_s (25 delta eta kappa_s/6 - 35 delta kappa_s/6
                 - 5 eta^2 kappa_a/3 + 95 eta kappa_a/6
                 - 35 kappa_a/6)
  + chi_s^2 (25 delta eta kappa_a/12 - 35 delta kappa_a/12
             - 5 eta^2 kappa_s/6 + 95 eta kappa_s/12
             - 35 kappa_s/12)
].
```

The cubic spin-induced contribution is

```text
E_SSS_SIM =
chi_a chi_s^2 (-2 delta^2 eta kappa_a - 5 delta^2 kappa_a
               + 6 delta eta kappa_s - 6 delta kappa_s - kappa_a)
+ chi_s^3 (-delta^3 kappa_a - delta^2 eta kappa_s
           - 9 delta^2 kappa_s/4 - delta kappa_a + kappa_s/4)
+ chi_a^2 chi_s (-6 delta kappa_a + 4 eta^2 kappa_s
                 + 12 eta kappa_s - 6 kappa_s)
+ chi_a^3 (-2 delta eta kappa_s - 2 delta kappa_s
           + 2 eta kappa_a - 2 kappa_a).
```

## Flux at Infinity

Appendix A writes

```text
F_infty = (32/5) eta^2 v^10 [
    F_NS(v) + F_SO(v) v^3 + F_SS(v) v^4
    + F_SSS(v) v^7 + F_Love(v) v^10
].
```

For this compact benchmark use

```text
F_infty = (32/5) eta^2 v^10 [
    1 + F_SS_SIM(v) v^4 + F_SSS_SIM v^7
    + (39/8) Lambda_tilde v^10
].
```

The `Lambda_tilde` term is an effective leading Love-number flux correction.
With Newtonian binding energy, it reproduces the leading
`-39 Lambda_tilde v^10/2` tidal phasing quoted in the paper. The full
`E_Love` and `F_Love` expressions are not printed in Appendix A.

The spin-induced quadrupole flux contribution is

```text
F_SS_SIM(v) =
chi_a chi_s (delta^2 kappa_a + 2 delta kappa_s + kappa_a)
+ chi_a^2 (delta kappa_a - 2 eta kappa_s + kappa_s)
+ chi_s^2 (delta kappa_a - 2 eta kappa_s + kappa_s)
+ v^2 [
    chi_a^2 (-127 delta eta kappa_a/16 + delta kappa_a/14
             + 43 eta^2 kappa_s/4 - 905 eta kappa_s/112
             + kappa_s/14)
  + chi_a chi_s (-127 delta eta kappa_s/8 + delta kappa_s/7
                 + 43 eta^2 kappa_a/2 - 905 eta kappa_a/56
                 + kappa_a/7)
  + chi_s^2 (-127 delta eta kappa_a/16 + delta kappa_a/14
             + 43 eta^2 kappa_s/4 - 905 eta kappa_s/112
             + kappa_s/14)
]
+ v^3 [
    chi_a^2 (4 pi delta kappa_a - 8 pi eta kappa_s + 4 pi kappa_s)
  + chi_a chi_s (8 pi delta kappa_s - 16 pi eta kappa_a + 8 pi kappa_a)
  + chi_s^2 (4 pi delta kappa_a - 8 pi eta kappa_s + 4 pi kappa_s)
].
```

The cubic spin-induced flux contribution is

```text
F_SSS_SIM =
chi_a chi_s^2 (13 delta^2 eta kappa_a/3 + 27 delta^2 kappa_a/16
               + 4 delta eta kappa_s/3 + 15 delta kappa_s/8
               + 3 kappa_a/16)
+ chi_a^2 chi_s (95 delta eta kappa_a/12 + 15 delta kappa_a/8
                 - 26 eta^2 kappa_s/3 + 25 eta kappa_s/6
                 + 15 kappa_s/8)
+ chi_s^3 (-7 delta eta kappa_a/4 + 5 delta kappa_a/8
           - 26 eta^2 kappa_s/3 - 3 eta kappa_s + 5 kappa_s/8)
+ chi_a^3 (29 delta eta kappa_s/6 + 5 delta kappa_s/8
           + 43 eta kappa_a/12 + 5 kappa_a/8).
```

## Absorbed Flux

Appendix A gives the absorbed flux structure. For this benchmark use

```text
dotM =
1/2 (9 H1E_bar eta^2 chi_a + 9 H1E eta^2 chi_s) v^15
+ 1/2 [
    (9 H1B eta^2 + 45 H1E eta^2/2
     + 9 H1E_bar delta eta^2/2 - 27 H1E eta^3) chi_s
  + (9 H1B_bar eta^2 + 45 H1E_bar eta^2/2
     + 9 H1E delta eta^2/2 - 27 H1E_bar eta^3) chi_a
] v^17
+ 18 H0 eta^2 v^18.
```

Appendix A prints `9 H0 eta^2 v^18`. The benchmark uses `18 H0 eta^2 v^18`
to implement the BBH electric-magnetic-duality convention. If the optional
spin-linear parameters `H1E`, `H1B`, `H1E_bar`, and `H1B_bar` are not supplied,
set them to zero.

## Frequency-Domain Use

Use the balance law to obtain the chirp rate:

```text
F_tot = F_infty + dotM
dt/dv = -M_sec d(e)/dv / F_tot
f = v^3 / (pi M_sec)
df/dt = [3 v^2 / (pi M_sec)] / [dt/dv]
```

Use a standard restricted-amplitude SPA waveform with

```text
A_N(f) = sqrt(5/24) Mc_sec^(5/6) pi^(-2/3) f^(-7/6) / dL_sec
dfdt_N = 96 eta v^11 / (5 pi M_sec^2)
A(f)   = A_N(f) sqrt(dfdt_N / dfdt)
h(f)   = A(f) exp[-i psi(f)]
```

Document in `notes.md` how you choose the integration convention for the SPA
phase. Constant and linear-in-frequency offsets are degenerate with `phic` and
`tc`.

Use the cutoff and taper rules specified in the prompt.
