# Validity Bench Dataset

Time-domain mismatch of NRHybSur3dq8 against SXS numerical relativity for aligned-spin quasi-circular BBH.

## Source

- **NR Database**: SXS Gravitational Waveform Database (https://data.black-holes.org)
- **Surrogate**: NRHybSur3dq8 (via `gwsurrogate`)
- **Reference**: Varma et al., Phys. Rev. D 99, 064045 (2019)

## Selection Criteria

- **Binary type**: BBH only (`SXS:BBH:*`)
- **Spins**: Aligned-spin (perpendicular spin components < 0.01)
- **Eccentricity**: Quasi-circular (e < 0.01)
- **Mass ratio**: q in [1, 10] (NRHybSur3dq8 validity range)
- **810 candidates** selected, **786 processed** (24 failed due to q > 10)

## Waveform Processing

For each simulation:

1. Load h22 from SXS at the highest available resolution level
2. Drop junk radiation (skip 100 M past `metadata.reference_time`)
3. Generate h22 from NRHybSur3dq8 at matching (q, chi1z, chi2z)
4. Align both waveforms:
   - Peak of |h22| at t = 0 (refined via 5-point quadratic fit)
   - Phase = 0 at peak
   - Consistent phase sign (dphi/dt has same sign near peak)
5. Interpolate both onto common uniform grid (dt = 0.1 M)
6. Compute time-domain mismatch: 1 − ⟨h₁|h₂⟩ / √(⟨h₁|h₁⟩⟨h₂|h₂⟩)

## Curated Benchmark Files

- **`validity_training.h5`** — 393 simulations (even indices after sorting)
- **`validity_validation.h5`** — 393 simulations (odd indices)

### HDF5 Structure

```
attrs:
    description, source_nr, source_surrogate, split, n_simulations

/q          — (N,) mass ratio
/chi1z      — (N,) z-component of primary spin
/chi2z      — (N,) z-component of secondary spin
/omega0     — (N,) reference orbital frequency (geometric units)
/mm_td      — (N,) time-domain mismatch against NRHybSur3dq8
/sxs_id     — (N,) SXS simulation identifier
```

## Train / Validation Split

Split by alternating sorted index: even indices go to training, odd to validation.

### Regeneration

```bash
conda activate ./envs/gwbench
python scripts/curate_validity_dataset.py
```

Note: requires internet access to download SXS waveforms and the NRHybSur3dq8 surrogate data.
