# Remnant Bench Dataset

Remnant properties (final mass, spin, kick) for quasi-circular BBH mergers from the SXS catalog, including precessing systems.

## Source

- **Database**: SXS Gravitational Waveform Database
- **URL**: https://data.black-holes.org
- **Catalog version**: v3.0.0 (2025-05-14)

## Selection Criteria

- **Binary type**: BBH only (`SXS:BBH:*`)
- **Eccentricity**: Quasi-circular (e < 0.01)
- **Spins**: All spin configurations (non-spinning, aligned, precessing)
- **3855 total candidates** passing filters

## Selection Method

Greedy coverage-optimized selection (following waveform_bench methodology):

1. Partition candidates into three tiers: non-spinning, aligned-spin, precessing
2. Allocate proportional quotas to each tier
3. Within each tier, greedily select simulations maximizing minimum distance in normalized parameter space, with a small recency bonus (newer SXS simulations preferred)
4. Training selected first; validation selected from remaining pool

## Curated Benchmark Files

- **`remnant_training.h5`** — 300 simulations
- **`remnant_validation.h5`** — 300 simulations

Generated by: `scripts/curate_remnant_dataset.py`

### HDF5 Structure

```
attrs:
    description, source, split, n_simulations

/q          — (N,) mass ratio
/chi1x      — (N,) x-component of primary spin
/chi1y      — (N,) y-component of primary spin
/chi1z      — (N,) z-component of primary spin
/chi2x      — (N,) x-component of secondary spin
/chi2y      — (N,) y-component of secondary spin
/chi2z      — (N,) z-component of secondary spin
/omega0     — (N,) reference orbital frequency (geometric units)
/Mf         — (N,) remnant mass (units of M)
/chif_mag   — (N,) remnant spin magnitude
/vf_mag     — (N,) remnant kick magnitude
/delta_Mf   — (N,) NR error in Mf (|highest_lev - second_highest_lev|)
/delta_chif — (N,) NR error in |chif|
/delta_vf   — (N,) NR error in |vf|
/sxs_id     — (N,) SXS simulation identifier
/lev_high   — (N,) highest resolution level used
/lev_low    — (N,) second-highest resolution level used
```

### Parameter Ranges

| Parameter | Range |
|-----------|-------|
| q | 1 – 20 |
| Mf / M | 0.88 – 1.00 |
| \|chif\| | 0.004 – 0.95 |
| \|vf\| | 0 – 0.013 |
