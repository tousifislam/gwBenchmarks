# Analytic Bench Dataset

Non-spinning BBH (2,2) mode waveforms from the SXS catalog, covering mass ratios q = 1 to 20.

## Source

- **Database**: SXS Gravitational Waveform Database
- **URL**: https://data.black-holes.org
- **Catalog version**: v3.0.0 (2025-05-14)

## Selection Criteria

- **Binary type**: BBH only (`SXS:BBH:*`)
- **Spins**: Non-spinning (|chi1|, |chi2| < 0.01)
- **Eccentricity**: Quasi-circular (e < 0.005)
- **Mass ratio**: q in [1, 20]
- **Per unique q**: lowest eccentricity preferred, then newest simulation
- **41 total simulations** selected

## Preprocessing

For each selected simulation:

1. Load the highest available Lev (resolution level)
2. Extract the (2,2) strain mode
3. Drop junk radiation (skip 100 M past `metadata.reference_time`)
4. Time-align so the amplitude peak sits at t = 0
5. Resample onto a uniform grid with dt = 0.1 M
6. Crop ringdown at t = +100 M past peak, cap total length at 10000 M
7. Store as complex time series (real and imaginary parts)

For non-spinning systems the coprecessing frame coincides with the inertial frame and h_{2,-2}(t) = conj(h_{2,2}(t)), so only h22 is stored.

## Curated Benchmark Files

- **`analytic_training.h5`** — 20 training waveforms
- **`analytic_validation.h5`** — 21 validation waveforms

Generated by: `scripts/curate_analytic_dataset.py`

### HDF5 Structure

```
attrs:
    dt_geometric = 0.1
    t_align = "h22_amplitude_peak"
    non_spinning = True
    split = "training" | "validation"

/sims/<SXS:BBH:nnnn>/
    attrs: q, eccentricity, chi1_mag, chi2_mag, lev_used, n_samples
    /t             — (N,) uniform time grid in units of M, peak at t=0
    /h22_real      — (N,) Re(h_{2,2}(t))
    /h22_imag      — (N,) Im(h_{2,2}(t))

/metadata/
    /sxs_id        — (N_sims,) simulation identifiers
    /q             — (N_sims,) mass ratios
    /eccentricity  — (N_sims,) reference eccentricities
```
