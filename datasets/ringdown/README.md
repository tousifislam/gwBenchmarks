# Ringdown Benchmark Dataset

Kerr quasi-normal mode (QNM) frequencies for gravitational perturbations (s = −2).

## Source

- **Authors**: Cook, Gregory B.
- **Reference**: Cook & Zalutskiy, Phys. Rev. D 90 (2014) 124021 ([DOI](https://doi.org/10.1103/PhysRevD.90.124021))
- **Zenodo**: [10.5281/zenodo.2650358](https://zenodo.org/records/2650358)

| `KerrQNM_07.h5` | n = 7 | 105 MB |

### Raw file structure

Each file contains: `/n{nn}/m{±mm}/{l,m,n}` → 2D array with columns:

| Column | Description |
|--------|-------------|
| 0 | Dimensionless spin `a/M` (0 to ~0.999999999) |
| 1 | `omega_real` — real part of QNM frequency (units of 1/M) |
| 2 | `omega_imag` — imaginary part (negative = damped) |
| 3 | `A_lm` real part — angular separation constant |
| 4 | `A_lm` imaginary part |
| 5+ | Spheroidal harmonic expansion coefficients `b_p` (real/imag pairs) |

## Curated Benchmark File

**`ringdown_benchmark.h5`** (98 MB) — consolidated dataset for the benchmark.

Generated by: `scripts/curate_qnm_dataset.py`

### Structure

```
/l{l}/m{±m}/n{n}/
    spin          — (N,) dimensionless spin a/M
    omega_real    — (N,) real part of QNM frequency
    omega_imag    — (N,) imaginary part of QNM frequency
    A_lm_real     — (N,) angular eigenvalue (real)
    A_lm_imag     — (N,) angular eigenvalue (imag)
```

### Coverage

| Parameter | Range |
|-----------|-------|
| l | 2 – 16 |
| m | −16 to +16 (where \|m\| ≤ l) |
| n (overtone) | 0 – 7 |
| a/M (spin) | 0 to ~0.999999999 (1062 points per mode) |
| **Total modes** | **2280** (285 per overtone) |

### Schwarzschild limit check (a = 0)

| Mode (l,m,n) | omega_R | omega_I |
|--------------|---------|---------|
| (2, +2, 0) | 0.37367168 | −0.08896232 |

## Train / Validation Split

Split by interleaving spin points: odd-indexed spin values go to training, even-indexed to validation. All 2280 modes appear in both sets.

| Split | File | Spin pts/mode | Total data points | Size |
|-------|------|---------------|-------------------|------|
| Training | `ringdown_training.h5` | ~531 | ~1.2M | 55 MB |
| Validation | `ringdown_validation.h5` | ~531 | ~1.2M | 55 MB |

Generated by: `scripts/split_train_val.py`

python scripts/curate_qnm_dataset.py --raw-dir . --output ringdown_benchmark.h5
