# Waveform Bench Dataset

Coprecessing-frame (2,2) mode waveforms from precessing quasi-circular BBH simulations in the SXS catalog.

## Source

- **Catalog**: SXS Gravitational Waveform Database v3.0.0
- **Frame**: Coprecessing frame via `strain.to_coprecessing_frame()`

## Parameter Space

| Parameter | Range | Description |
|-----------|-------|-------------|
| q | [1, 8] | Mass ratio |
| chi1x, chi1y, chi1z | \|chi1\| ≤ 0.8 | Primary spin components |
| chi2x, chi2y, chi2z | \|chi2\| ≤ 0.8 | Secondary spin components |
| omega0 | varies | Reference orbital frequency (geometric units) |

## Selection

Greedy coverage-optimized selection with tiered allocation:
- 10 non-spinning (q only)
- 40 aligned-spin (q, chi1z, chi2z)
- 200 precessing (q, chi1xyz, chi2xyz)

Filters: q ∈ [1, 8], |chi| ≤ 0.8, eccentricity < 1e-3, 5 ≤ N_orbits < 40, remnant data available.

## Preprocessing

- Load highest available Lev
- Transform to coprecessing frame
- Drop junk radiation (reference_time + 100M buffer)
- Resample to uniform dt = 0.1M
- Peak-align: coprecessing |h22| peak at t = 0
- Phase convention: phase(h22_copr) = 0 at t = 0
- **No cosine taper applied**
- Cap ringdown at t = +100M, total length ≤ 10000M

## Curated Benchmark Files

- **`waveform_training.h5`** — 250 simulations
- **`waveform_validation.h5`** — 250 simulations

Generated by: `scripts/curate_waveform_dataset.py`

### HDF5 Structure

```
attrs:
    description, source, split, n_simulations, dt_geometric, t_align, phase_convention

/sim_XXXX/
    attrs: sxs_id, q, chi1x, chi1y, chi1z, chi2x, chi2y, chi2z,
           omega0, n_pts, lev_used, wf_length,
           nr_fd_mm_M{40,80,120,160,200}, nr_fd_mm_combined (if available)
    /t         — (N,) time in geometric units (M), peak at 0
    /h22_real  — (N,) Re(h22_copr)
    /h22_imag  — (N,) Im(h22_copr)

/metadata/
    /sxs_id, /q, /chi1x, ..., /chi2z, /omega0  — (N_sims,) parameter arrays
    /chi_eff, /chi_p  — (N_sims,) derived quantities
    /wf_length  — (N_sims,) waveform duration in M

/nr_errors/
    /sxs_id  — (M,) simulation IDs with NR errors
    /fd_mismatch_M{40,80,120,160,200}  — (M,) FD mismatch at each total mass
    /fd_mismatch_combined  — (M,) max FD mismatch across masses
```

## NR Resolution Errors

Computed by comparing coprecessing-frame h22 between highest and second-highest resolution level (Lev) for each simulation. Frequency-domain mismatch is evaluated at M_tot = {40, 80, 120, 160, 200} M_sun using the aLIGO PSD (via PyCBC). The combined NR error is the maximum across all total masses.

### Regeneration

```bash
conda activate ./envs/gwbench

```
