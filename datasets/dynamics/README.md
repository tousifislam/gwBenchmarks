# Dynamics Bench Dataset

Eccentric BBH orbital dynamics — eccentricity e(t), relativistic anomaly zeta(t), and PN frequency parameter x(t) — generated with SEOBNRv5EHM.

## Source

- **Model**: SEOBNRv5EHM (via `pyseobnr`)
- **Reference**: Gamboa et al., arXiv:2412.12823 (2024)

## Parameter Space

| Parameter | Range | Description |
|-----------|-------|-------------|
| q | [1, 6] | Mass ratio |
| chi1z | [-0.6, 0.6] | Primary spin z-component |
| chi2z | [-0.6, 0.6] | Secondary spin z-component |
| e0 | [0.001, 0.5] | Initial eccentricity |
| zeta0 | [0, pi] | Initial relativistic anomaly |
| omega0 | [0.0075, 0.0085] | Initial orbital frequency (geometric units) |

## Sampling

Latin Hypercube Sampling in 6D parameter space, with separate seeds for training (42) and validation (137).

## Curated Benchmark Files

- **`dynamics_training.h5`** — 250 simulations
- **`dynamics_validation.h5`** — 250 simulations

Generated by: `scripts/curate_dynamics_dataset.py`

### HDF5 Structure

```
attrs:
    description, source, split, n_simulations, sampling

/sim_XXXX/
    attrs: q, chi1z, chi2z, e0, zeta0, omega0, n_pts
    /t     — (N,) time in geometric units (M)
    /e     — (N,) eccentricity evolution
    /zeta  — (N,) relativistic anomaly evolution
    /x     — (N,) PN frequency parameter x = <omega>^(2/3)

/metadata/
    /q, /chi1z, /chi2z, /e0, /zeta0, /omega0  — (N_sims,) parameter arrays
```

### Regeneration

```bash
conda activate ./envs/gwbench
python scripts/curate_dynamics_dataset.py
```

Note: requires `pyseobnr` with SEOBNRv5EHM support.
